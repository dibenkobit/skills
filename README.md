# skills

A collection of AI agent skills by [@dibenkobit](https://github.com/dibenkobit).

## Install

```bash
npx skills add dibenkobit/skills
```

## Available Skills

| Skill | Type | Description |
|-------|------|-------------|
| [atomic-commits](skills/atomic-commits/) | Command | Split staged changes into atomic commits on demand |
| [atomic-commits-philosophy](skills/atomic-commits-philosophy/) | Guideline | Always commit atomically — no invocation needed |

## Skill Types

**Command skills** — invoke explicitly ("commit atomically", "/atomic-commits")

**Guideline skills** — always active, shape agent behavior by default
